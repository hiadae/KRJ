TARGET STRATEGY

3層ターゲット戦略

窪田様の洞察を元に、ターゲットを3層に分けて最適なアプローチを設計する。

LAYER A

家購入希望者

「いつ家を買うべきか」で迷っている層。対面で窪田様に会うことで初めて動く。

📍 最適戦略：対面接触
サンパーク催事・セミナー → その場でLINE登録 → 教育動画で信頼構築 → 個別相談
  • サンパーク催事でライトなテーマ（お小遣いの作り方、教育資金、車の買い替え計画）で接触
  • 「今買う vs 3年後に買う」シミュレーションアプリでその場で具体的な数字を提示
  • 「高い金利でも今買った方が得」という逆転提案が窪田様の独自性
  • 教育資金と住宅購入を同時並行で実現する具体的プランの提示
💰 成約ライン
住宅ローン紹介 → 金融機関パートナーからの紹介手数料 15〜35万円/件
⚠️ 裏切りリスク高：勉強会で情報だけ取り、他のハウスメーカーに行くケースが多い。セミナー中に「ここでしか得られない価値」を感じさせ、個別相談に確実に繋ぐ設計が必須
LAYER B

家計不足層

月収不足 → 貯蓄取崩し → ローン利用開始。「あと3万円あれば…」が口癖の層。

📍 最適戦略：対面 + 広告のハイブリッド
対面催事 or LINE/メタ広告 → LINE登録 → 家計改善シミュレーション → 個別相談
  • 最も確度の高い層 — 裏切る余裕がない。来れば確実に相談につながる
  • 月3〜5万円の改善で人生が変わる。「大きな節約（住まい・車・保険）」の提案が刺さる
  • 家計改善シミュレーションアプリでその場で「変わる」実感を持ってもらう
  • ローン借換・保険見直しで月々の返済を数万円単位で削減可能
  • 山口県で年間約1,000人が破産 → その手前にいるB層を救えば6割は「治療」可能
💰 成約ライン
家計改善コンサル → ローン借換・保険見直し紹介 → 紹介手数料 10〜25万円/件
改善実績が口コミとなり、追加紹介の連鎖が生まれやすい層
💡 対面催事にも広告にも反応する二刀流ターゲット。催事では「節約セミナー」、広告では「あと3万円で生活が楽になる方法」の訴求が有効
LAYER C

借金膨張層

借金が雪だるま式に膨張中。勉強会には絶対に来ない（他人に借金を知られたくない）。

📍 最適戦略：「破産する前に」コンテンツ → 秘密厳守の個別相談
Google検索広告 → 「破産する前に」LP → LINE登録 → 秘密厳守の個別相談予約
  • 対面には絶対来ない — デジタル→秘密の個別相談が唯一の導線
  • 借金膨張プロセス: 月3万不足 → 年40万 → 5年で300万超の借金
  • 「ブラックではない＝まだ借入可能」段階での介入がポイント
  • 借金を全部整理 → ローンで低金利に組み直し → 月収プラスに転換 → 伴走支援
  • 窪田様の見立て: 破産前に来れば約60〜70%は救済可能
💰 成約ライン
債務整理・ローン再構成 → 金融機関からの紹介手数料 20〜40万円/件
1件あたりの単価が最も高い。ローン組成額が大きいため金融機関からの手数料も大きい
🔒 プライバシー最優先。サンパーク店舗ではなく、オンライン or 完全個室での相談が必須。「誰にも知られずに、破産せずに済む方法があります」が最重要メッセージ

対面催事でA層・B層をどう切り分けるか

同じサンパーク催事に来るA層・B層を混在させず、セミナーテーマ自体をフィルターとして活用する。
FP・保険業界で最も実績のある手法。

A層向けセミナー（月1〜2回・土日）

🏠「失敗しないマイホーム購入術」

  • ✦ 「お小遣いUP講座」「教育資金の貯め方」
  • ✦ 「今買う vs 3年後に買う」シミュレーション実演
  • ✦ 「車の計画的な買い替えで月3万浮く方法」
想定来場者: 家族連れ、30〜40代、住宅購入検討中
根拠: FP住宅相談ネットワークでは「住宅購入セミナー」で年間400家族が参加。テーマ名がフィルターとなり、住宅検討者が自然に集まる
B層向けセミナー（月1〜2回・平日）

💰「あと3万円を生み出す家計術」

  • ✦ 「ローン借換で月3万円削減する方法」
  • ✦ 「保険料を半額にする見直し術」
  • ✦ 「桁違いの節約ヒント — 住まい・車・保険」
想定来場者: 主婦層、40〜50代、家計に課題を感じている
根拠: 保険代理店の実績では、「家計見直し」テーマで新規顧客獲得率が大幅向上。平日開催は主婦・パート層の参加率が高い

📋

催事から個別相談までの一気通貫フロー

催事参加の条件として「LINE登録 + アンケート回答」を設定。催事後はAIがアンケート内容に基づきパーソナライズした追メッセージを自動生成・配信し、個別相談へ誘導する。

1
催事参加条件：LINE登録 + アンケート回答
受付でQRコードを読み取り → LINE友だち追加 → リッチメニューからアンケートに回答。
アンケート例（選択式3問・30秒で完了）:
Q1. 今日のセミナーに参加した理由は？
　□ マイホーム購入を検討中　□ 家計の見直しをしたい　□ 将来のお金が不安　□ なんとなく気になった
Q2. 今一番気になるお金のことは？
　□ 住宅ローン　□ 教育資金　□ 保険料　□ 毎月の生活費　□ 借金・ローン返済
Q3. 個別相談（無料）に興味はありますか？
　□ ぜひ相談したい　□ 少し興味がある　□ 今は大丈夫
→ 回答内容でA層（住宅系）/ B層（家計系）タグが自動付与
2
催事参加（対面で窪田様の信頼を構築）
セミナー or 無料相談会に参加。窪田様の人柄と専門性を直接体感してもらう。
シミュレーションアプリを使ってその場で「数字」を見せ、具体的な変化をイメージさせる。
3
催事後：AIがパーソナライズした追メッセージをLINEで自動配信
催事終了後、その日のうちにすぐ配信。アンケート回答 + 参加テーマに基づき、一人ひとり異なる内容のフォローメッセージをAIが自動生成 → LINE配信。窪田様の手間はゼロ。
A層への追メッセージ例
「本日はセミナーにご参加ありがとうございました。ご質問いただいた教育資金と住宅購入の両立について、○○様の場合のシミュレーション結果をまとめました。よろしければ個別にお話しさせてください。」
B層への追メッセージ例
「本日はお忙しい中ありがとうございました。アンケートで保険料が気になるとのことでしたので、見直しで月○万円削減できた事例をお送りします。詳しくは無料の個別相談でお話しできます。」
4
個別相談予約 → 成約
当日中の追メッセージに反応した方を個別相談に誘導。LINE上で予約完結。
対面の熱が冷めないうちに、その日のうちにフォローが届くことで行動率が大幅に上がる。
反応がなかった方にも、3日後・1週間後にセグメント別の教育コンテンツ（YouTube動画等）を段階配信し、じわじわ温める。

この仕組みのポイント: 催事で得た「対面の熱量」が冷める前に、その日のうちにAIパーソナライズで一人ひとりに届ける。
大量の参加者にも手書きのような個別対応が即日完了 — しかも窪田様の作業時間はゼロ。

致命的

広告の遷移先が正しく設定されていない

自社制作の2キャンペーン（消化額¥52,345・1,906クリック）の遷移先がAIチャットサービスのトップページになっており、クリックした全員が離脱しています。また、野中広告はChatworkのファイルダウンロードリンクで、ログインしていない一般ユーザーはアクセスできません。

改善: 遷移先URLを正しいLP（ランディングページ）に差し替えるだけで、月間1,906クリックをLINE友達追加に転換できます。LP制作＋ホスティング（Webサーバー上への公開）を弊社で対応します。
高評価

自社制作の広告クリエイティブは優秀

自社制作キャンペーンのCPC（クリック単価）は¥26〜31で、モード社制作（¥46〜56）の約半額。広告のデザイン・コピーは人の関心を引く力があり、「返済が減らない方へ」「匿名で返済診断」といった訴求は的確です。

強化: このクリエイティブ力を活かすために、受け皿となるLP・導線の整備が最優先。既存のクリエイティブを再利用し、正しいLPに接続するだけで成果が見込めます。
要改善

モード社のパターン①② — 遷移先サイトのコンテンツが薄い

kurashino119.infoへ正しく遷移していますが、サイト自体のコンテンツが薄く、来店や予約への明確な導線がありません。392クリック（¥19,989）を獲得しましたが、来店転換率は0%です。

改善: kurashino119.info にLP機能（相談予約フォーム、LINE友達追加ボタン、窪田所長の実績・人柄が伝わるコンテンツ）を追加。広告からの流入を逃さない受け皿に。
方向性◎

「破産する前に」コンテンツ — 競合不在の好機

①②③向け・❹❺❻❼❽向けのクリエイティブは「債務整理」「返済改善」に特化しており、山陽小野田エリアでこのキーワードのGoogle検索広告は未出稿。競合がいない市場です。

強化: 既存クリエイティブを活かし、Google検索広告を追加。「山陽小野田 借金相談」「住宅ローン 返済 減らない」で検索する「今まさに困っている人」にリーチできます。
要改善

Webサイト — ドメイン4つ中2つがアクセス不可

kurashino119.com と .net がDNSエラーでアクセスできません。さらに、おのだサンパーク公式サイトのテナント紹介ページに掲載されているURLが .com（アクセス不可）のため、サンパーク来店客がWebサイトに辿り着けません。

改善: サンパーク公式への掲載URLを .jp に変更依頼。不要なドメインは整理し、広告・印刷物のURLを .jp または .info に統一。
要改善

LINE友達追加後の「教育→予約」導線

LINE友達は2,048人（認証済みアカウント）と十分な規模。しかし登録後のステップ配信や自動応答が不足しており、友達追加→放置→離脱の流れになっています。

改善: 登録後72時間以内にステップ配信（自己紹介→相談事例→無料診断の案内→予約促進）を自動構築。セグメント別（住宅/家計/債務）の配信で予約率を最大化。
